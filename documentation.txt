🏗️ Sistema interno de gestión – Empresa de 
construcción modular

Objetivo:
Centralizar proyectos (obras), tareas, objetivos y 
usuarios, con roles por proyecto.

User {
  _id
  name
  email
  passwordHash
}

Project {
  _id
  name
  description
  location
  startDate
  endDate
  status
}

ProjectMember {
  _id
  userId
  projectId
  role : ENUM = (READ, READWRITE)
}

Task {
  _id
  projectId
  title
  description
  assignedTo = array
  status ENUM = (PENDIENTE, PROGRESO, BLOQUEADA, PAUSADA, FINALIZADA)
  priority ENUM = (BAJA, MEDIA, ALTA, URGENTE)
  dueDate
}

TaskComment {
  _id
  taskId
  userId
  comment
  createdAt
}

Usuario
  ↓
ProjectMember (rol por proyecto)
  ↓
Proyecto
  ↓
Tareas
  ↓
Comentarios
